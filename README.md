# github-issue-list

## 本番 URL

issue 一覧ページ  
https://github-issue-list.netlify.app/issues/

## 使用技術

- Nuxt.js (Composition API)
- TypeScript
- Jest / Vue Test Utils
- Tailwind CSS

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# start mock server
$ npm run json
```
